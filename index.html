Through exploratory data analysis (EDA) and interactive visualizations, I identify key insights into income trends, expense distributions, and financial behaviour over time.

Strava Analysis

This project offers a deep dive into my cycling performance using data extracted from Strava via its API. By leveraging Python, Pandas, and data visualization libraries, I analyze ride details such as distance, elevation, speed, and trends over time. I uncover patterns in my training progress, compare ride efficiency, and identify areas for improvement.

Tech used:

  • Python
  • Strava API
  • Seaborn
  • Pandas
  • Plotly

Google Data Analysis

As part of my Google Data Analytics Certification, this case study explores user behavior patterns from wearable technology data, focusing on customer fit, sleep quality, and potential mental health concerns. Using Python, Pandas, and data visualization techniques, I analyze smartwatch data to uncover trends related to physical activity, rest cycles, and 'bed rotting' time—prolonged periods of inactivity that may indicate mental health struggles.

Tech used:

  • Python
  • Pandas
  • Google Career Certificate
